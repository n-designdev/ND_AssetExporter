ND_AssetExporter Help

2021, 0608

ツールUIの説明

  • オプション
    • project environment load

      • 環境設定を読み込んだmaya batchを使用します。

    • camera scale override

      • カメラスケールをオーバーライドします。オフの場合シーン内の値をそのまま使います。

    • abc step override

      • abcステップをオーバーライドします。

    • debug

      • 出力先がpublish/test_charにエクスポートされます。テストしたいときにお使いください。

    • bake animation

      • キーフレームが打たれたオブジェクトをベイクします。

    • frameRangeCustom

      • フレームレンジを上書きします。

    • FrameHundle

      • フレームハンドルを設定します。オフの場合±5フレームが自動で設定されます。

  • D&Dエリア

    • ここにファイルをドラッグ・アンド・ドロップすることでプロジェクトやショット、シーケンス情報を解析し、それを元にshotgunから情報を取得します。

    正しく取得された場合、shotgun上のアセット情報がテーブル化され表示されます。

  • パス

    • D&Dエリアにドロップしたファイルのパスが表示されます。

  • Exportボタン
    • Local Export

      • そのままエクスポートを実行します。

    • Submit to Deadline

      • deadlineで実行します(試験中)。

ShotGridの説明


  • 1 Category List

    • chara, bg, prop, cameraのアセットがリストアップされます。

  • 2 Asset Name

    • ここで登録された名前がリストアップされます。アセットの名前は出力ファイルの名前にもなります

  • 3 Asset Path

    • アセットのパスを登録してください。

  • 4 Namespcae

    • ネームスペースを登録してください。プロップなどで後ろに数字が付く場合は正規表現を用いてください。

  • 5 TopNode

    • トップノードを登録してください。

  • 6 Export type

    • エクスポートタイプを選択してください。anim, abcのどちらかを選択できます。

  • 7 ABC Export

    • abcで出力する場合にはここにエクスポート対象を登録しください。複数登録する場合は「,」で区切ってください。

  • 8 Animation Export List

    • animで出力する場合にはここにエクスポート対象を登録してください。複数登録する場合は「,」で区切ってください。

  • 9 Sequence

    • 登録されたシーケンスのシーンでは、Assetをシーケンス単位で出力したい場合に登録してください。対象のシーケンスでは、shot側で設定されているAssetに加えここで登録したアセットも追加でリストアップされます。

  • shotページでは、以下のReferenced Assetsを設定してください。設定されたアセットがND_AssetExporterにリストアップされます。

Tips

Chara Importerで設定されるフレーム設定はCameraを書き出した際にシーン内のminFrame, maxFrameを出力します。

Current Refresh Toolボタンを押すことで、パブリッシュディレクトリのcurrentが最新かチェックします。最新でない場合は出力された最新のバージョンをコピーします。

リストアップされるCamera はAssetとして登録されたものでは無く、デフォルトで入っているものです。シーン内のカメラを検索し、authographicsではないカメラを簡易的にコピーします。複雑なカメラを使う場合はアセットとして登録してください。

Shotgunでは少なくとも、shot, Asset Path, Namespace, Top Node, Export Type Anim Export Listを登録してください。
